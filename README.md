# SimpleCalculator
A lightweight Java-based command-line calculator with core arithmetic functions and optional extended features.

## Core Features
- Basic arithmetic: Addition, Subtraction, Multiplication, Division, Modulo
- Essential math functions: Square root, Factorial, Absolute value, Average calculation
- Input validation: Prevents division by zero, square root of negative numbers
- Interactive menu: User-friendly command-line interface

## Optional Extended Features (Can be added)
1. Support for floating-point numbers (instead of only integers)
2. Batch calculation with multiple numbers
3. History record of calculation results
4. Support for parentheses in complex expressions
5. GUI (Graphical User Interface) instead of command-line
6. More math functions (logarithm, trigonometric functions: sin/cos/tan)

## How to Run
1. Compile the code: `javac SimpleCalculator.java`
2. Run the program: `java SimpleCalculator`
3. Follow the on-screen menu to select operations
